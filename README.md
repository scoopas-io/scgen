# Musikkatalog vorbereitet für (scoopify – KI Musik Streaming)

## Projektinfo

## Technologie-Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Lokale Entwicklung

```sh
# Repository klonen
git clone <YOUR_GIT_URL>

# In das Projektverzeichnis wechseln
cd <YOUR_PROJECT_NAME>

# Abhängigkeiten installieren
npm i

# Entwicklungsserver starten
npm run dev
```

## Deployment

Das Projekt wird automatisch bei jedem Push in den Hauptbranch deployed.

## Custom Domain

Unter Project > Settings > Domains kann eine eigene Domain verknüpft werden.
